plan2:
reset(): prev_energy_cost → prev_total_energy

  _get_rewards() 完全替换，实现了 Plan 2 的三层结构：
  - r_energy：能耗节约，clip 到 [-1, 1]
  - r_task：任务拒绝惩罚 / 完成奖励
  - r_shared = alpha_energy * r_energy + beta_task * r_task（全员共享）
  - r_lb = -|selected_util - mean_util|（仅被选中的 local agent 额外加 gamma_lb * r_lb）

  超参默认值：alpha_energy=1.0, beta_task=0.5, gamma_lb=0.1，已在 __init__ 中注册，消融时直接传参即可

  schedulers/marl/hier_marl/HierMARL.py — The coordinator class implementing:
  - select_action: runs global + all local actors, returns integer actions
  - add: converts integer actions to one-hot, stores global transition always + selected local's transition only
  - learn: updates global agent (neighborhood critic with only the selected local's obs/act) and each local agent
  independently
  - _update_global / _update_local: DDPG-style critic + actor updates with Gumbel-softmax
